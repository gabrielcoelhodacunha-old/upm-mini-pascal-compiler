# Mini Pascal Compiler
## About
Simplified **Python** compiler for the [Mini Pascal] language.

## Why
Developed as a requirement for the **compiladores (compilers)** subject during my 6th semester of Computer Science.

## Tools used
- [Jupyter Notebook]
- [rply]
- [re]
- [networkx]
- [matplotlib]
- [itertools]
